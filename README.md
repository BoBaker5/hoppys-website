# Hoppy's Christmas Tree Farm

The website for Hoppy's Christmas Tree Farm — a family-run cut-your-own farm in
North Stonington, Connecticut.

🌐 **Live site:** _(your Vercel URL will go here once deployed)_

## What's in here
- `index.html` — the whole website (one page)
- `images/` — the logo and farm photos

It's a plain static website: just HTML, CSS, and images. No build step, nothing
to install. Vercel serves it exactly as-is.

## Editing the site
You can edit `index.html` right here on GitHub (pencil icon → make your change →
Commit). Vercel automatically re-deploys within a minute of every commit.

Common things to update:
- **Hours / opening day** — search the file for "November 27" and "weekends 9"
- **Address** — search for "108 Ryder Rd"
- **Email** — search for "hoppytreeroh@gmail.com"
- **Contact form** — search for "YOUR_FORM_ID" (see below)

## Contact form
The form shows a friendly fallback (pointing to email/Facebook) until it's
connected. To make it deliver messages: sign up free at https://formspree.io,
create a form, and replace `YOUR_FORM_ID` in `index.html` with the ID it gives
you (looks like `https://formspree.io/f/abcdwxyz`).
